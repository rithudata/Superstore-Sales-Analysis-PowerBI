# Superstore-Sales-Analysis-PowerBI
SQL and Power BI project analyzing sales performance using a star schema model.

## Project Overview
This project analyzes sales performance data from a fictional superstore 
using SQL for data cleaning and modeling, and Power BI for visualization.

## Tools Used
- **SQL (MySQL)** — data cleaning, joins, aggregations
- **Power BI** — interactive dashboard
- **Star Schema** — fact and dimension table design

## What I Analyzed
- Sales performance by category, segment, and region
- Year-over-year trends
- Top and bottom performing states
- Profit vs. discount relationship

## Key Insights
- Technology category had the highest sales
- West region outperformed other regions
- Heavy discounting in Furniture reduced profit margins

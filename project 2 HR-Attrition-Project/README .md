# 🧑‍💼 HR Attrition Analysis Project

## 📌 Project Overview
This project analyzes employee attrition data to identify key factors influencing employee turnover in an organization.  
The goal is to understand **why employees leave the company** and identify patterns using SQL and Power BI.

---

## 🛠 Tools Used
- SQL (MySQL)
- Power BI

---

## 📊 Problem Statement
Employee attrition is a major challenge for organizations.  
This project aims to analyze HR data and find:
- Which factors increase attrition
- Which employee groups are at higher risk
- What business actions can reduce attrition
